# 后端选项

## leancloud

### appId

- 类型：`string`
- 默认值：`""`

Leancloud 的 appId

### appKey

- 类型：`string`
- 默认值：`""`

Leancloud 的 appKey

### spa

- 类型：`boolean`
- 默认值：`false`

是否在单页应用中使用。如果是，则需要先调用 `Modis.initAv` （不可重复调用）初始化 Leancloud sdk 后再使用 `new Modis`
