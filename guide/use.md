# 使用

## 引用

### NPM

```
npm i @mrzhiin/modis
```

### 浏览器

```html
<script src="path/modis.min.js"></script>
```

或者分别引入资源

```html
<script src="//cdn.jsdelivr.net/npm/leancloud-storage@3.11.0/dist/av-min.js"></script>
```

```html
<script src="path/modis.slim.min.js"></script>
```

## 调用

```html
<div id="modis"></div>

<script>
  new Modis({
    el: "#modis",
    appId: "",
    appKey: ""
  });
</script>
```
